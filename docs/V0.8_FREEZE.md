# 镜隙之间 V0.8 Freeze

## 版本定位

V0.8 是"美学导出版"。

本版本在 V0.7 重新显影的基础上，补全"把时间帧带走"的能力，让用户可以将单帧或多帧导出为具有《镜隙之间》视觉气质的 PNG 图片。

## 核心完成项

### 1. 单帧 PNG 导出
- FrameDetailOverlay 增加 PNG 导出入口
- 使用 html-to-image 将帧详情渲染为 PNG
- 输出 2x pixelRatio 高清图片
- 内容包含帧编号、日期时间、摘要、原文节选、标签、tone、品牌署名

### 2. 多帧合集长图
- ArchivePanel 支持勾选多帧导出 PNG
- 选择 1 帧导出单帧图
- 选择 2–8 帧导出合集长图
- 选择超过 8 帧禁用导出并提示"长图≤8帧"
- 新增 FrameCollectionImageExport 组件

### 3. 桌面 / 移动端导出体验
- 桌面端直接通过 `<a download>` 下载
- 移动端优先使用 navigator.share
- 分享不可用或失败时 fallback 下载
- 文件命名为 jingxi-frame-YYYY.MM.DD-HHMM.png 等格式

### 4. 视觉系统
- 导出图保留暗房 / 纸纹 / 底片封套感
- paper-grain 纹理兼容 html-to-image
- 三套主题通过 inline CSS 变量注入，避免导出图割裂

### 5. 验收结果
- 帧详情单张 PNG PASS
- 档案馆选中 1 帧 PASS
- 档案馆选中 3 帧合集长图 PASS
- 档案馆全部记录概览 PNG PASS
- 桌面端直接下载 PASS
- 选中 8 帧合集长图 PASS
- 选中 9 帧禁用提示 PASS
- PNG 文件头验证通过
- npm run build 通过
- tsc --noEmit 零错误
