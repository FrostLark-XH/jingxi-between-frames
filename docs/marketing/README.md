# Marketing Assets

This folder stores shareable visual material outside the runtime app.

Naming:
- `xhs-*.png`: Xiaohongshu carousel images, 3:4 vertical.
- `store-*.png`: app store / install preview assets.
- `source-*`: editable source or raw capture used to compose exports.

Rules:
- Use real product screenshots as the primary subject.
- Keep generated or decorative elements secondary.
- Do not store secrets, user data, or private test records here.
