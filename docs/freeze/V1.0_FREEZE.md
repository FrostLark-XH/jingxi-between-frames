# 镜隙之间 V1.0 Freeze

## 版本定位

V1.0 是"内测安全版"。

在 V0.9 "移动端 PWA 部署与导出精修版"基础上，补齐内测前必需的三项安全能力（数据备份恢复、API 限流、匿名埋点）和轻量反馈入口，并修复 P0 移动端主题撕裂 bug。

## 核心完成项

### 1. 数据管理（DataManager 组件）

- 新增 `components/DataManager.tsx` — 右侧滑入面板，仿 ArchivePanel 的 Framer Motion 动效
- 导出备份：`triggerBackupDownload()` 生成 `jingxi-backup-{date}.json`，UTF-8 BOM，含软删除帧及完整 AI 字段
- 导入恢复：`<input type="file">` → JSON.parse → `validateBackupData()` 结构校验（数组 + id/content/date）→ 覆盖确认 → `IMPORT_FRAMES` action 完全替换
- 清空数据：危险按钮 → 键入"确认清空"二次确认 → `CLEAR_ALL_FRAMES` action
- 新增 `IMPORT_FRAMES` / `CLEAR_ALL_FRAMES` reducer actions（`hooks/useAppState.ts`）
- 入口：RecordingRoom 底部"数据管理"链接 + FilmPage header"数据管理"按钮

### 2. API 限流（lib/rateLimit.ts）

- 模块级 `Map<string, number[]>` 滑动窗口限流，Edge Runtime 兼容
- 频率限制：60 秒窗口，每 key 最多 10 次请求
- 内容限制：develop-frame 单次 ≤ 3000 字，summarize-day 单次 ≤ 20 帧
- 429 响应：`{ error: "暗房暂时忙碌，请稍后再试。", retryAfter: N }`
- 限流在 API 处理最前端，不影响帧保存（帧先落盘，AI 后台处理）

### 3. 匿名埋点（lib/analytics.ts）

- `track(event, metadata?)` 通过 `navigator.sendBeacon()` 发送
- 6 个事件点：`app_open` / `first_frame_created` / `frame_developed` / `film_opened` / `png_exported` / `backup_exported`
- `app_open` 含 `deviceType`（mobile/desktop）和 `displayMode`（browser/standalone/fullscreen）
- 不上传用户原文、摘要、标签、任何 PII
- `NEXT_PUBLIC_ANALYTICS_ENDPOINT` 未配置时静默跳过

### 4. 反馈入口

- AppShell 右上角条件渲染 `<a>` 链接"反馈体验"
- URL 由 `NEXT_PUBLIC_FEEDBACK_URL` 环境变量控制，未配置则完全隐藏
- `target="_blank"` + `rel="noopener noreferrer"` 安全打开

### 5. P0 主题撕裂 bug 修复

- **根因**：CSS 变量双重写入源 — CSS `[data-theme]` 选择器 vs `applyThemeToDocument()` 的 `style.setProperty()`（内联样式）。内联样式优先级更高，一旦短暂不同步即产生暗亮撕裂
- **修复**：
  - `lib/themes.ts`：`applyThemeToDocument()` 删除所有 `style.setProperty()`，仅设 `data-theme` 属性 + `colorScheme`。CSS 变量由 `[data-theme]` 选择器唯一控制
  - `app/globals.css`：`:root` 及每个 `[data-theme]` 块加入 `color-scheme`；FOUC 防护改用 `data-theme-ready`
  - `app/layout.tsx`：inline script 重写 — 主题白名单校验 + `data-theme-ready` + 错误回退时设 fallback
  - `hooks/useTheme.tsx`：`<meta name="theme-color">` 值与 inline script 对齐，移除残留 `data-theme-loading` 引用
- **验证**：Playwright 3 主题 × 冷启动/PWA 重开/切换 全场景通过；真机（iOS Safari）验证通过

## 当前边界

- 无账号系统、无云同步、无后端数据库
- 暂不上架应用商店
- 暂不处理中国大陆访问稳定性
- 暂不做 PDF 导出
- 暂不做语音输入
- 暂不做 AI 聊天 / 心理分析
- 数据保存在 localStorage
- 仅中文

## 技术栈

- Next.js 15 App Router + React 19 + TypeScript 5.8
- Tailwind CSS v3.4 + Framer Motion 12
- Three.js（shader 背景）
- html-to-image（PNG 导出）
- localStorage 全量持久化
- Cloudflare Pages + Vercel 双平台部署
- Playwright 1.59（E2E）
