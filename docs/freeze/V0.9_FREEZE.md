# 镜隙之间 V0.9 Freeze

## 版本定位

V0.9 是"移动端 PWA 部署与导出精修版"。

在 V0.8"美学导出卡片版"基础上，完成 PWA 化、双平台部署、导出卡片视觉重构、以及多项移动端修复。

## 核心完成项

### 1. PWA 支持

- 新增 `public/manifest.webmanifest` — PWA 配置文件
- 新增 `public/icons/icon-192.png` / `icon-512.png` / `apple-touch-icon.png`
- `app/layout.tsx` 添加完整的移动端 web app meta 标签：
  - `apple-mobile-web-app-capable` / `apple-mobile-web-app-status-bar-style`
  - `theme-color` / `format-detection`
  - OpenGraph / Twitter Card 社交预览标签
- 全屏 PWA 模式下原生应用体验

### 2. 双平台部署

- Vercel：`jingxi-between-frames.vercel.app`（自动从 GitHub 部署）
- Cloudflare Pages：`jingxi-between-frames.pages.dev`
- `.github/workflows/deploy.yml` — GitHub Actions CI/CD
- `wrangler.toml` — nodejs_compat 兼容标志
- CLI 设置 GitHub Secrets（PyNaCl 加密 + GitHub API）
- Ubuntu Linux runner 绕过 Windows 下 `@cloudflare/next-on-pages` 兼容问题

### 3. 导出卡片视觉重构

- 信息层级反转：用户原文升为视觉主体（20px 全色），AI 摘要降为辅助（16px 60% 透明度）
- `stripIndent()` 去除全角空格首行缩进，导出图左侧对齐严格一致
- 标签改为淡底无边框胶囊（`bg=accent/5%`），圆角 4px
- 帧编号减小（11px mono），不再抢眼
- 移除 footer 品牌名称，右对齐仅保留"时间帧档案卡"
- 多帧长图同步应用以上所有视觉规则
- AI 输出清洁：`sanitizeText()` 清洗 LLM 常见标点错误（。，→， 等 8 条规则）

### 4. 保存 PNG / 分享 PNG 拆分

- `lib/exportImage.ts`：新增 `shareBlob()` + `canShare()`，移除旧的 `shareOrDownload()`
- `FrameDetailOverlay.tsx`：导出行为"保存 PNG"+"分享 PNG"两个同级按钮
- `ArchivePanel.tsx`：单帧、多帧选中、总览，三种场景均拆分保存/分享按钮
- `canShare()` 不可用时分享按钮自动隐藏

### 5. ArchivePanel 多帧长图导出

- 单帧选中 → `FrameImageExport` 单帧卡片
- 多帧选中（2-8 帧）→ `FrameCollectionImageExport` 多帧长图
- 总览导出：1 帧用单帧，≥2 帧用长图
- `renderToBlob()` 统一接口，`html-to-image` 离屏渲染

### 6. 移动端 PNG 导出空白修复

- 导出 DOM 从 `visibility:hidden` / `position:absolute` 改为 `overflow:visible` on-screen 层
- 修复 Safari / iOS 下 `foreignObject` SVG 渲染兼容性问题
- `hexToRgba()` 统一色彩处理，避免 Safari 不支持 `#RRGGBB` 在 SVG 中

### 7. MD / TXT UTF-8 BOM 乱码修复

- JSON/MD/TXT 导出统一使用 `triggerDownload()` 的 `addBOM` 参数
- 文件名格式化：单帧 `jingxi-frame-{date}-{time}.{ext}`，多帧 `jingxi-archive-{date}.{ext}`
- `getExportFilename()` 统一各格式命名逻辑

### 8. 主题启动闪色 / FOUC 修复

- `app/layout.tsx` 内联 `<script>` 在 `<html>` 渲染前注入 `data-theme` 属性
- 防止亮/暗主题切换时的首帧闪白

### 9. AI 文本清洗

- `services/ai/index.ts`：`sanitizeText()` 函数
- 清洗常见 LLM 标点错误：`。，`→`，`、`，。`→`。`、`。。`→`。`、`，，`→`，` 等 8 条规则
- real / mock 双路径均经过 sanitize

## 当前边界

- 暂不处理中国大陆访问稳定性
- 暂不上架应用商店
- 暂不做账号系统、云同步、数据库
- 暂不做 PDF 导出
- 暂不做语音输入
- 暂不做 AI 聊天 / 心理分析
- 数据仍保存在 localStorage
- 仅中文

## 技术栈

- Next.js 15 App Router + React 19 + TypeScript 5.8
- Tailwind CSS v3.4 + Framer Motion 12
- Three.js（shader 背景）
- html-to-image（PNG 导出）
- localStorage 全量持久化
- Cloudflare Pages + Vercel 双平台部署
- Playwright 1.59（E2E + 导出图生成）
