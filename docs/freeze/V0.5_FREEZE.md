# 镜隙之间 V0.5 Freeze

## 版本定位

V0.5 是"帧回看闭环版"。

本版本完成核心闭环：

记录一帧 → 显影一帧 → 归档一帧 → 回看一帧 → 编辑一帧 → 导出一帧

## 核心完成项

### 1. 显影反馈系统
- MemoryInput 显影动画最终版
- 去除 brightness / opacity 压暗文字的问题
- 文字全程清晰
- 使用双云团暖光从上至下穿透输入容器
- 使用 blur、grain、柔性拖尾与暖色染色
- 从"扫描线"调整为"显影液流过相纸"的视觉隐喻

### 2. 时间帧系统
- 统一"帧"作为最小记录单位
- 移除"片段"作为顶层时间尺度
- 时间结构统一为：年 / 月 / 日 / 帧
- 日视图展示时间导轨、卷首卡与帧列表

### 3. 帧详情与回看闭环
- FrameDetailOverlay 支持查看、编辑、标签管理
- 支持单帧 JSON / Markdown / TXT 导出
- 新增 UPDATE_FRAME 数据流
- 支持回收站与 7 天软删除

### 4. 视觉系统升级
- 字体系统完成
- 三套主题完成深化
- 动效氛围完成收束
- safe-area 与响应式适配完成
- semantic color tokens 基本贯通
- paper-grain 与 WebGL 背景整合

### 5. 移动端与性能
- safe-area 全覆盖
- min-h-dvh 降级
- prefers-reduced-motion 支持
- 移动端 hover 禁用
- WebGL 主题切换不重建
- TypeScript 零错误
- Next build 通过

## 当前边界

- 无真实 AI，仅 mock / 规则模拟
- 无语音输入
- 无 PDF / 图片导出
- 无后端
- 无账号
- 无云同步
- 数据保存在 localStorage

## 下一阶段建议

V0.6：真实显影引擎。

V0.6 只做真实 AI 的单帧摘要、标签生成、日卷主线。

不要做 AI 聊天、心理分析、复杂知识库、语音、PDF、图片导出。
