# 镜隙之间 V0.6 Freeze

## 版本定位

V0.6 是"真实显影引擎版"。

本版本在 V0.5"帧回看闭环版"的基础上，完成真实 LLM 显影能力接入。

## 核心完成项

### 1. 真实单帧显影

- 新增 `/api/ai/develop-frame`
- 用户保存一帧时，优先调用真实 LLM
- 生成显影摘要、标签、关键词、tone
- 返回 `provider` / `model` 信息
- AI 失败时 fallback 到 mockProvider
- 原文始终保存，不被 AI 失败阻断

### 2. 真实日卷主线

- 新增 `/api/ai/summarize-day`
- 基于当天所有帧生成今日主线
- 生成关键词与轻量回看提示
- DaySummaryCard 支持 `aiDaySummary`
- API 失败时回退规则生成逻辑

### 3. AI 服务层

- 扩展 `services/ai/types.ts`
- 新增 `prompts.ts`
- 新增 `realProvider.ts`
- `mockProvider` 兼容新结构
- `index.ts` 统一 real → mock fallback 编排

### 4. 安全与环境变量

- 使用 `LLM_API_KEY` / `LLM_BASE_URL` / `LLM_MODEL`
- API Key 只在服务端使用
- 前端不直接调用第三方 LLM
- `.env.local` 不提交
- realProvider 不打印 API Key

### 5. DMXAPI 实测

- `develop-frame` 已返回 `provider: real`
- `summarize-day` 已返回 `provider: real`
- model: `DeepSeek-V3.2-Thinking`
- mock fallback 路径完整

## 当前边界

- 无语音输入
- 无 PDF / 图片导出
- 无 AI 聊天
- 无心理分析
- 无账号系统
- 无数据库
- 无云同步
- 数据仍保存在 localStorage
